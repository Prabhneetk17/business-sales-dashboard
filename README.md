# Business Sales Dashboard | Microsoft Excel
## Overview

This project is an interactive Microsoft Excel dashboard built using the Sample Superstore dataset. It analyzes business performance across different regions, product categories, customer segments, and shipping modes to generate actionable business insights.

---

## Dataset

- **Records:** 9,995 Orders
- **Time Period:** 2014–2017
- **Regions:** East, West, Central, South

---

## Tools & Excel Functions Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- SUMIFS
- COUNTIFS
- XLOOKUP / VLOOKUP
- SUMPRODUCT
- Conditional Formatting
- Data Validation
- Interactive Dashboard Design

---

## Dashboard Features

- Executive KPI Dashboard
- Sales Analysis
- Profit Analysis
- Regional Performance
- Monthly Sales Trend
- Category-wise Performance
- Shipping Analysis
- Dynamic Filters
- Lookup Table Integration

---

## Business Questions Answered

- Which region generates the highest sales?
- Which region is the most profitable?
- Which product categories underperform?
- How do monthly sales change over time?
- Which customer segments contribute the most revenue?

---

## Key Insights

- Technology is the most profitable product category.
- Furniture generates comparatively lower profit despite strong sales.
- The West region consistently performs well across key metrics.
- Interactive filters allow users to explore trends across multiple business dimensions.
